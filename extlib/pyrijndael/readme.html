pyRijndael
Version:1.3
Author:Jeffrey Clement
Status:Stable

A pure python implementation of the AES encryption standard, Rijndael.

Project Description

This is a pure python implementation of the Rijndael encryption algorithmwith a really easy to use interface. If you need easy secure encryption foryour python software look no further. Being that this is written in Python beware that it is really quite slow for large data sets.

*2003/09/03* - fixed bug for future Python versions (2.4+)

Requirements

  • Python 2.2+
