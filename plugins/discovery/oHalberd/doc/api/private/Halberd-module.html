Package Halberd

HTTP load balancer detector module.
Submodules
  • clientlib: HTTP/HTTPS client module.
  • clues: Clue management package
    • analysis: Utilities for clue analysis.
    • Clue: Clue generation module.
    • file: Utilities for clue storage.
  • conflib: Configuration file management module.
  • crew: A work crew is instantiated passing a ScanTask object as a parameter, thus defining the target and the way the scanning should be done.
  • logger: Logger singleton.
  • reportlib: Output module.
  • ScanTask: Scanning tasks.
  • shell: Provides scanning patterns to be used as building blocks for more complex scans.
  • util: Miscellaneous functions.
